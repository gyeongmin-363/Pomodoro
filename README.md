# 날씨앱 + 뽀모도로

### 목적
뽀모도로를 이용하여 공부하는 이들은 보통 바깥을 잘 보지 못하는 경우가 많다. 
화면을 통해서라도 바깥 상황을 알리고자 날씨앱과 뽀모도로를 합친다.

### 기능
- 뽀모도로 : 어플 종료 및 폰 종료 시에도 타이머 흘러감 + 다른 화면 전환 시 차단효과
- 날씨 : 어플 배경 이미지로 현재 지역의 시간(일출, 일몰), 날씨, 습도 등을 반영한 배경을 출력
  - 미세먼지, 우천 유무 등에 따라 쉬는시간 행동 추
- 방 : 활동 유무와 공부 시간을 웹소켓(혹은 supabase realtime)으로 보임
  - supabase realtime은 free tier에선 한번에 최대 200명까지의 연결만 가능.
  - 웹소켓을 하기엔 aws 무료 체험도 소모함.
  - 그러므로 일단 방 기능을 후순위로 미룸
