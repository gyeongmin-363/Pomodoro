# 🎶 뽀모도로 × 음악 보상 앱

## 1. 핵심 컨셉
- **집중(뽀모도로 세션)** → **보상(코인 획득)** → **음악 요소 해금 & 조합**  
- 사용자의 집중 결과물이 **음악으로 변환**됨 → 독특한 성취감 제공  
- **기본 세트(필수 비트/드럼/코러스)**는 항상 제공  
- 나머지는 **세션 보상 코인으로 해금**  
- **미리듣기 지원** → 구매 전 사운드 확인 가능  

---

## 2. 보상 루프 (사용자 흐름)
1. **세션 시작**  
   - 기본 세트 사운드만 사용 가능  
   - 집중 완료를 위해 타이머 실행  

2. **세션 종료 → 보상 획득**  
   - 세션 시간(분 단위)에 따라 코인 지급  
   - 코인으로 새로운 사운드/악기/효과음 구매 가능  

3. **음악 요소 해금**  
   - 상점에서 원하는 사운드를 **미리듣기** 후 구매  
   - 구매한 사운드는 "내 컬렉션"에 영구 등록  

4. **음악 조합 / 트랙 생성**  
   - 내 컬렉션을 사용해 **나만의 루프/트랙** 제작  
   - 제작한 트랙은 게시글로 공유 가능  

5. **커뮤니티 공유**  
   - “오늘 3세션 성공 → 드럼+보컬 추가, 이런 트랙 완성 🎶”  
   - 게시글에는 **플레이어 미니 UI** 삽입 (재생/멈춤)  
   - 좋아요(👍), 리액션(🔥👏🎧) 가능  

6. **딥링크 활용**  
   - 게시글 공유: `myapp://post?id=123` → 해당 트랙 게시글 바로 열기  
   - 앱 미설치 시: Netlify에 호스팅된 **웹 미리듣기 페이지**로 연결  
   - SNS/Twitter/카톡 등에서 자연스럽게 확산  

---

## 3. 기능 요약

### 🎵 음악 보상 시스템
- **기본 제공**: 드럼, 베이스, 단순 보컬 등 최소 3~4개  
- **해금 요소**: 기타, 신스, 효과음, 리드 보컬, 코러스 등  
- **코인 시스템**: 세션 25분 → 코인 25개  
- **상점 기능**: 사운드 미리듣기 → 코인으로 구매  

### 📝 커뮤니티(게시판)
- **게시글 = 음악 트랙 + 성취 기록**  
  - 예: “오늘 50분 집중 성공 → 코인 50 획득, 새로운 기타 해금 🎸”  
- 리액션/댓글: 간단한 이모지/짧은 응원  
- 정렬: 최신 / 인기 / 나의 트랙  

### 🔗 딥링크
- **내 게시글 공유**: 친구가 클릭하면 해당 트랙 재생 가능  
- **방문 유저**: 앱 없으면 웹 미리듣기 제공 → 앱 설치 유도  

---

## 4. 장점
- **동기부여**: 세션 성공 ↔ 보상 해금이 직접 연결  
- **차별성**: 기존 뽀모도로 앱과 달리 **음악 창작 + 수집 경험** 제공  
- **디자인 부담↓**: 음악 중심, UI는 심플 카드 + 플레이어 위주  
- **커뮤니티 활성화**: 경쟁 대신 자랑/공유 → 긍정적 분위기  

---

## 5. 확장 아이디어
- 희귀 사운드(전설 악기, 특수 FX) → 장기 목표  
- 시즌 한정 사운드 → FOMO 요소 제공  
- 그룹 게시판 → 공동 세션으로 합주곡 완성
